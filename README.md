# Job Market & Salary Insights (2025)

## Overview
This project analyses 2025 Data Science, AI, and Machine Learning job roles to
understand how salary varies by role, experience level, and technical skills.

## Key Questions
- Which roles pay the highest salaries?
- How does experience influence compensation?
- Which skills provide the strongest salary leverage?

## Methodology
- Data cleaning and standardisation using Python
- Feature engineering for experience and skill indicators
- Exploratory data analysis
- Interactive dashboard development in Power BI

## Key Insights
- ML and AI roles command the highest median salaries
- Salary growth accelerates significantly at senior and executive levels
- ML and AI skills provide additional salary leverage beyond Python

## Tools Used
Python, Pandas, Power BI

## Limitations
- Dataset is static and does not represent real-time postings
- Skills inferred from job titles only
